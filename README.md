# ICMBioIlheus
Gerador de relatórios fotográficos para uso pelos ATAs do NGI ICMBio Ilhéus

# Relatório Fotográfico Ambiental

Sistema simples para registro de vistorias ambientais com:

- Geolocalização
- Mapa com KML
- Upload de fotos
- Exportação KMZ com imagens

## Como usar

1. Coloque o arquivo KML na mesma pasta
2. Abra `index.html` via servidor local ou Netlify

## Exportação KMZ

O KMZ gerado pode ser aberto no:
- Google Earth
- QGIS
- ArcGIS
